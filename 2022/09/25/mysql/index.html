mysql属于关系型数据库，数据库中数据用关系表示。一个关系对应一张数据表。

数据库操作语句

数据定义语言DDL

  • 创建表 CREATE TABLE table_name
  • 创建视图 CREATE VIEW view_name AS SELECT ···
  • 创建索引 CREATE INDEX index_name ON table_name(column_name)
  • 创建唯一索引 CREATE UNIQUE INDEX index_name ON table_name(column_name)
  • 创建聚簇索引 CREATE CLUSTERED INDEX index_name ON table_name(column_name)
  • 删除 DROP table_name/index_name/view_name
  • 修改表 ALTER TABLE table_name ADD column e

数据操纵语言DML

  • 插入语句 INSERT INTO table_name(column1,column2,···) VALUES (e1,e2,···)
  • 删除语句 删除表 DELETE FROM table_name / 删除数据 DELETE FROM table_name WHERE ···
  • 修改语句 UPDATE SET table_name WHERE ···

数据查询语言DQL

  • 查询语句
    • 单表查询
    • 多表连接 -WHERE

数据控制语言DCL

  • 授权给用户 GRANT ALL/privileges on database.table/database.* TO user@ipAddress
  • 收回用户权限 REVOKE ALL/privileges on database.table/database.* FROM user@ipAddress
